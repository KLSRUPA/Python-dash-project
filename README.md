# Swedish Population Dashboard

This is a data visualization dashboard built using **Dash** and **Plotly** to explore the Swedish population distribution by age and sex for the year 2022 and earlier. The dashboard allows the user to interactively select the year and sex to view the population distribution across different age groups.

## Features

- **Interactive Bar Chart**: Displays the population distribution by age for men or women.
- **Dropdowns for Year and Sex**: Users can select a specific year and sex (men or women) to view the relevant data.
- **Dynamic Updates**: The bar chart updates automatically when users change the selected year or sex.

## Requirements

- Python 3.x
- Required libraries:
  - **Dash** (for the dashboard)
  - **Plotly** (for data visualization)
  - **Pandas** (for data manipulation)
